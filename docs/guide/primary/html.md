# HTML :star:
HTML（HyperText Markup Language），超文本标记语言。主要掌握其各个标签的用法及标签属性，以及HTML5的新特性。

## 标签
### 基本标签
``` html
<html></html>      定义HTML文档

<head></head>      文档的信息

<meta>             HTML 文档的元信息

<title></title>    文档的标题

<link>             文档与外部资源的关系

<style></style>    文档的样式信息

<body></body>      可见的页面内容

<!--…-->           注释
```
### 文本标签
``` html
<h1></h1>          标题（h1~h6）

<b></b>            定义粗体文本

<strong></strong>  粗体字(强调) 

<i></i>            斜体字 

<ul></ul>          无序列表 

<ol></ol>          有序列表

<li></li>          列表项目

<a></a>            链接

<br>               换行

<p>                段落
```
